Introduction to Thymeleaf

Thymeleaf is a modern server-side Java template engine for web and standalone environments. It allows you to render dynamic content in HTML templates and can be easily integrated with Spring Boot.

1. Thymeleaf Syntax

Thymeleaf uses special attributes to manipulate HTML content dynamically:

  • th:text: Replaces text inside an element.
  • th:if: Conditionally renders an element if the expression evaluates to true.
  • th:each: Iterates over collections and renders elements for each item.
  • th:href: Dynamically sets the value of the href attribute.
  • th:src: Dynamically sets the value of the src attribute.

Example Code


        <div th:text="${message}">This is a default message</div>

This will replace the content of the <div> tag with the value of the message attribute passed from the controller.

2. Thymeleaf Expressions

Thymeleaf expressions are evaluated within the HTML tags, making them easy to use for dynamic content.

  • ${expression}: Retrieves the value of the expression.
  • #{message}: Retrieves messages from properties files.
  • *{expression}: Refers to an object’s properties.

Example Code:


        <span th:text="${'Welcome, ' + user.name}">User</span>

This will display "Welcome, John" if the user.name value is "John".

Conditional Rendering with Thymeleaf

Thymeleaf provides th:if and th:unless for conditionally rendering elements based on the evaluated expression.

1. Using th:if

The th:if attribute will render the element only if the condition evaluates to true:


        <div th:if="${user.active}">User is active</div>

This element will only be rendered if the user.active property is true.

2. Using th:unless

The th:unless attribute will render the element if the condition evaluates to false:


        <div th:unless="${user.active}">User is not active</div>

This element will only be rendered if the user.active property is false.

Iterating Over Collections

You can iterate over a collection (list, array, etc.) using the th:each attribute.

Example: Iterating Over a List of Users


        <ul>
          <li th:each="user : ${users}" th:text="${user.name}"></li>
        </ul>

This will iterate over the users list and render each user's name in a list item.

Example: Iterating Over a Map


        <ul>
          <li th:each="entry : ${map}" th:text="'Key: ' + ${entry.key} + ', Value: ' + ${entry.value}"></li>
        </ul>

This will iterate over a map and display both the key and the value in a list item.

Form Binding with Thymeleaf

Thymeleaf allows you to bind forms to Java objects. Here’s an example of how to use th:field to bind form fields to a model attribute.

Example: Binding a Form


        <form th:action="@{/user/save}" th:object="${user}" method="post">
          <label for="name">Name</label>
          <input type="text" id="name" th:field="*{name}" />

          <label for="email">Email</label>
          <input type="email" id="email" th:field="*{email}" />

          <button type="submit">Save</button>
        </form>

This will bind the form fields to the name and email properties of the user object.

    <p>Hello, <span th:text="${user.name}">User</span>! Welcome back.</p>
</div>

    <p>Please log in to access your account.</p>
</div>
